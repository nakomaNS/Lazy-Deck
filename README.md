## Sobre o Projeto

O **Lazy Deck** é uma ferramenta de "idle farming" projetada para quem quer completar insígnias ou vender cartas no mercado sem precisar instalar gigabytes de jogos. A aplicação escaneia sua conta, identifica quais jogos ainda possuem cartas para dropar e **simula a execução do jogo, fazendo com que a Steam libere as cartas.**

Ideal para farmar cartas de jogos que você comprou em bundles e nunca abriu.

## Funcionalidades

* **Sincronização Inteligente:** Faz login seguro através do navegador integrado (cookies) e varre sua página de insígnias para criar uma fila de execução automática apenas com jogos que possuem drops restantes.

* **Fila Tripla:** Para quem tem pressa, este modo executa até 3 jogos simultaneamente, triplicando a velocidade de obtenção de cartas (uso por conta e risco, mas altamente eficiente).

* **Gerenciamento de Fila:** Acompanhe o progresso em tempo real com timers precisos, e minimize o app para a bandeja do sistema (Tray) sem interromper o processo.

* **Leve e Seguro:** Utiliza processos mínimos que consomem pouquíssima memória RAM e CPU, diferente de abrir o jogo real.
